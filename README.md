Artificial Intelligence-Powered Genetic Risk Predictor for Lifestyle Diseases
Developed an AI-driven predictive system to assess genetic and lifestyle-based risks for common lifestyle diseases such as diabetes, cardiovascular disorders, and hypertension. The project leverages machine learning algorithms implemented in Python using Scikit-learn to analyze genetic markers, demographic data, and lifestyle factors including diet, physical activity, BMI, and family medical history.

Performed data preprocessing, feature engineering, and normalization to improve model reliability and reduce bias. Trained and evaluated multiple ML models (such as Logistic Regression, Random Forest, and Support Vector Machines) to identify the most accurate predictor, achieving 92% prediction accuracy. Model performance was validated using metrics like accuracy, precision, recall, and confusion matrix.

The system provides early risk prediction, enabling proactive lifestyle changes and preventive healthcare planning. This project demonstrates strong skills in machine learning, data analysis, healthcare analytics, and model evaluation, with a focus on real-world medical applications and ethical AI usage.

Model Details
Features used: Age, BMI, Gender, Sleep, Exercise, Junk food intake, Smoking, Drinking.

Models used:
Logistic Regression
Random Forest
Support Vector Machine (SVM)
